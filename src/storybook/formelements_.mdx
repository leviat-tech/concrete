# Form Elements

All form inputs have the ability to wrap them selves in a form element component.
By default form inputs will use the `wrapFormInputs` property in the concrete options unless the `noWrap` property has been set on the form input.

```javascript dark
// entry.js
app.use(Concrete, {
  wrapFormInputs: true // default value
});
```
